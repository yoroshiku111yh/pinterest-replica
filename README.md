
# Replica Pinterest

- A project code by Nestjs (Backend) and Nextjs (Frontend) 
- For course backend Nodejs by Framework Nestjs, upgrade with Frontend Nextjs




## Backend
- /backend
- Backend :  yarn start:dev
- Port : 8080
- Api list ( Swagger ) : http://localhost:8080/swagger/

## Frontend
- /front-end
- Frontend : yarn dev
- Port : 3000

## Database
- /database/option-sql
- Port : 3306
- table name ( using in .env ) : pinterest_capstone_41

## Tools
#### Frontend
- React hook form
- Typescript
- Tailwind css
- lodash
- Nextjs

#### Backend
- Nestjs
- JWT Passport
- Typescript

## Requirements
- Node : version > 20
